Portfolio

News site (The Guardian API)

This mini-project was created to meet the requirements for one of my final year modules. The brief was to create a website which could dynamically display articles using an API from a news website. The user should be able to click to read the articles; view different categories; be shown the top stories; and perform a search, both online and offline. The site was also to be a single page application.
I used The Guardian's API and JavaScript to complete this, and used Bootstrap for styling. I also used the OpenWeatherMap API to add an extra feature allowing users to check the weather in their city or search for another city.

Travel Lab

Travel Lab is a web application I created in fulfilment of one of my final year modules. The assignment brief was to allow a customer to review and update their saved quotation for a package holiday. The data was loaded from a JSON file using AJAX. As the technology for this application is client-side, any updates made to the details weren't required to be submitted back to the server.

If you want to give the application a go, here's some booking references you can use that will return data: TLB008515, TLB682554, TLB951164 (any name will work).
